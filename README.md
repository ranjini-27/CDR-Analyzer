# CDR Forensic Analyzer

A web-based forensic investigation dashboard for analyzing Call Detail Records (CDR), communication behavior, and cellular tower data.

## Features

- CDR CSV upload and parsing
- Session-based case management
- Contact frequency analysis
- Behavioral anomaly detection
- Network relationship visualization
- Daily activity timeline
- Hourly communication patterns
- Radar-based anomaly scoring
- Tower location estimation
- Sample downloadable datasets
- Interactive forensic dashboard UI

## Technologies Used

- Python
- Flask
- Pandas
- NumPy
- HTML/CSS/JavaScript
- Chart.js
